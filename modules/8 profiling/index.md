# Profiling

Als laatste onderdeel van deze cursus staat de opdracht "Profiling". Deze is 3 punten waard binnen de cursus en telt gewoon mee. De opdracht is wel gemarkeerd als extra onderwerp omdat de rest van de cursus niet bouwt op deze kennis. Als je een opdracht moet overslaan is deze opdracht een goede keuze. Je mist daarmee dan wel enkele punten.

Per goed onderbouwde concrete optimalisatie krijg je 0,5 punt. Het maximum is 3 punten.
